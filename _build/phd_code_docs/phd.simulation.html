class phd.simulation.simulation.Simulation(simulation_name='simulation', colored_logs=True, log_level='debug', **kwargs)¶

Bases: object

Marshalls the simulation.

This class runs the simulation and all necessary outputs. It takes an integrator and simulation time and builds an output directory where files will be saved.
