Installation
============

The relevant packages can be installed from the Github `here <https://github.com/hegdesahil/phd-code>`_. You will also need to install the `Computational Geometry Algorithms Library (CGAL) <https://www.cgal.org/>`_ via::

	brew install cgal

Once you have that installed, you can navigate to the home directory for the phd code and use the command::

	pip install -e .
